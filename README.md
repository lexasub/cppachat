# cppachat
chat on tcp posix(serv + cli) functional style
this is multiclient chat.
using:
cli:
scommad - run some server command
*msg - send to all
cid$msg - send msg to id - socket
serv:
exit - off server and say it all
*msg - send to all
cid$msg - send msg to id - socket
somecommand - run command on server
